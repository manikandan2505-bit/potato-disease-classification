# Potato Disease Classification

An AI-powered web application that detects potato leaf diseases using a Convolutional Neural Network (CNN).

## Technologies Used

- TensorFlow / Keras
- FastAPI
- React (Vite)
- Python
- Axios

## Features

- Upload potato leaf images
- Predict disease
- Confidence score
- Disease description
- Treatment recommendations

## Classes

- Healthy
- Early Blight
- Late Blight
